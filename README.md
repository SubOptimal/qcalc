# Quick Calculator
A java TUI calculator.
<br />
This project is a TUI calculator that can evaluate elmentary functions.
It should work on Linux/Mac/ Anything that can run sh scripts and java.
<br />
**Syntax:** qcalc *[Optional -flag]* function
